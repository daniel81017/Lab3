This is an interactive map of the University of Toronto's St. George (downtown) campus. Features of the map include a boundary polygon, bike routes, and suggested study spots. Upon the user's first click into campus, the camera uses the flyTo() function to zoom into campus slightly. Gone is the campus colour shade after an initial click. The points utilize the flyTo() function's pitch and bearing features, allowing users to visualize the buildings in 3D form. Further features (boundary and bike lane lines) dissppear upon this user command. All code was classified as a single variable so as to facilitate a reload of the map without the need for the user to refresh the page.
